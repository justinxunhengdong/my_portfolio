Food Waste Management App

In Their Shoes

Designing a website using and manipulating one or more APIs from the State Library of Queensland

Made for a Interactive Technology course over one semester at the University of Queensland.

Designed in a group of five over one semester. Received a high distinction for this project. Served as project lead in this group.

Brainstormed as to how the API data could be used to create an interactive, engaging learning tool. Decided on creating a hidden-object game inspired by games from my childhood such as Professor Layton, Where in the World is Carmen Sandiego and The Crimson Room. Acted as the game's designer, designed the game from its scoring system, gameplay, educational value, visual design, and potential to add complexity in future levels. This was especially difficult as the game dynamically created new levels, much like the game GeoGuessr, and the data in the API was not always consistent or usable. As such, we had to research and analyse puzzle and question design on a very granular level.

Along with designing the game, I handled supervision of team-members, deadlines, and front-end directions. I also programmed the majority of the back-end in PHP, by fetching the data and processing it using RegEx (Regular Expressions), and devising a system to analyse the data's viability. I also had to optimise the back-end due to the size of the database. I also aided the team member assigned to the game logic, who used JavaScript and JQuery to send data to the front-end.
